Property of block tags.

  • Block elements always come to new line.
  • Take the space from extreme left to extreme right.
  • we can also provide them a custom width and height.
  • We can also provide custom margin and padding
  • Block tag can be changed to inline type using display

Property of inline tag

  • Come in same line
  • Width can not be change
