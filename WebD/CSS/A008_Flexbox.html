This is a very powerfull tool.

  • It is 1D Layout method, either in row or column.
  • Whenever we want to change any element in Flex then we put the display property as flex.
  • By default the direction of flex is row. we can use flex-direction property to change to column.
    Flex Direction can be ->
    • row
    • row-reverse
    • column
    • column-reverse
  • Using Flex we can create a good responsive design for differenet screen.
  • Flex-wrap is propert which don'ta llow the inner content width or height change, the content comes to next line soon the screen won't fit for it, till the space is available.
  • Justify-content -> it place according to the main axis(depends on flex direction) and values availble are start, end, center, space can be added
  • align-item -> by default value is vertical
