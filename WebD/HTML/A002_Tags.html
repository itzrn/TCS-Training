Tag is a Markup entity. Using Tag we can provide structure, meaning, format to content.
Sementic Tag -> h1 to h6, p, a, ul, table, img, form, input
Structure Tag -> html, head , body, header, main, aside, footer, article, section, nav
Formatting -> b, i, u, strong, sup, sub, pre, em, code, br(is a self closing tag)
